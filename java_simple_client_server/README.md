# Java Simple Client-Server

Простой многопоточный TCP клиент-сервер на Java. Сервер принимает несколько соединений одновременно, читает входящие сообщения и отключает неактивных клиентов по таймауту.

Проект состоит из двух модулей IntelliJ IDEA: **Server** и **Client**.

Пакет: `ru.er_log`. Java 8+.
