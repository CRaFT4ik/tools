# AI Photo Description Generator

Инструмент для массового анализа фотографий из путешествий с помощью параллельных ИИ-агентов. На выходе — Markdown-файл с текстовыми описаниями каждого кадра, группировкой по сценам и рекомендациями по выбору лучших снимков.

Протестировано на ~3000 фото из трёх поездок.

## Состав

- `compress.py` — сжатие оригиналов в миниатюры для анализа (требует `Pillow`)
- `agent_prompt_template.md` — промпт-шаблон для суб-агентов
- `INSTRUCTION_FOR_ORCHESTRATOR.md` — полная инструкция по оркестрации процесса
- `recommendations-example.md` — пример итогового файла
