# Рекомендации по фотографиям: Турция 2025

Этот документ содержит анализ фотографий из поездки в Турцию (апрель–май 2025).
Фотографии сгруппированы по сценам — кадры, сделанные с разницей в несколько секунд,
объединены в одну группу. Для каждой группы выбран лучший кадр, остальные помечены
как дубли или неудачные. Цель — помочь быстро отобрать лучшие снимки и предоставить
описание содержимого для использования в статьях о путешествиях.

> Время в названии файлов Pixel записано в UTC. Местное время пересчитано с поправкой +3 часа (GMT+3, Турция).
