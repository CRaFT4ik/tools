# Tools

Сборник утилит и скриптов.

| Папка | Описание |
|-------|----------|
| [ai_photo_description_generator](ai_photo_description_generator/) | Массовый анализ travel-фотографий параллельными ИИ-агентами. Скрипт сжатия + оркестрация. |
| [all_to_one_printer](all_to_one_printer/) | CLI-утилита на Java — склеивает содержимое файлов по маске в один файл. |
| [java_simple_client_server](java_simple_client_server/) | Простой многопоточный TCP клиент-сервер на Java. |
| [linux_things](linux_things/) | Systemd-сервисы и скрипты для Linux (Opera VPN proxy, драйвер Wi-Fi). |
